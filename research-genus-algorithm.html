An Efficient Genus Algorithm Based on Graph Rotations

With Alexander Metzger, I co-authored a paper presenting PAGE, an algorithm that determines the orientable genus of an arbitrary finite connected graph using graph rotation systems. Computing graph genus is notoriously hard, and many existing algorithms struggle with the placement of bridges; PAGE sidesteps that difficulty entirely.

The algorithm is designed to be useful in practice: it is simple to implement, it outputs the faces of an optimal embedding, it produces a proof certificate that can be independently verified, and it can be used to obtain both upper and lower bounds on the genus. We illustrate it by determining previously unknown genera — most notably showing that the $(3,12)$-cage has genus $17$.

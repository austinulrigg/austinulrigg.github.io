# austinulrigg.github.io

Personal academic website for Austin Ulrigg, built with Jekyll and hosted with GitHub Pages at <https://austinulrigg.github.io/>.

The site includes:

- About, CV, teaching/service, writing, and tutoring pages
- PDF writing samples and an undergraduate honors thesis
- A math tutoring inquiry page and contact form

## Local Development

Install the Ruby dependencies, then run Jekyll locally:

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

The local preview is available at <http://localhost:4000/>.

## Theme

This site is based on the Academic Pages template, which is derived from Minimal Mistakes.
